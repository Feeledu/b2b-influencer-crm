B2B Influencer CRM - Frontend Test

Frontend is running on: http://localhost:8080

Backend API is running on: http://localhost:8000

API Documentation: http://localhost:8000/docs

Test Instructions:

  1. Open http://localhost:8080 in your browser
  2. You should see the landing page with "InfluenceHub" branding
  3. Click "Get Started" to go to the auth page
  4. Since we're in demo mode, you'll be automatically logged in as a demo user
  5. Navigate to the Dashboard to see metrics
  6. Go to "Discover" to see the influencer discovery page with mock data
  7. Go to "My Influencers" to see your saved influencers (empty initially)

Expected Behavior:

  • ✅ Landing page loads with hero section
  • ✅ Authentication works in demo mode
  • ✅ Dashboard shows mock metrics and data
  • ✅ Discover page shows 3 mock influencers with AI analysis
  • ✅ My Influencers page shows empty state initially
  • ✅ "Add to My List" functionality works (saves to local state)
